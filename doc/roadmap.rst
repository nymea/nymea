Roadmap
=======

The roadmap outlines the major themes planned for upcoming nymea releases.

Short term
----------

* Migrate existing plugins to the latest SDK revisions.
* Improve test coverage for JSON-RPC edge cases.

Mid term
--------

* Provide official Docker images with long-term support.
* Expand the set of certified devices.

Long term
---------

* Introduce a declarative rule builder for complex automation scenarios.
* Publish a stable REST gateway to complement JSON-RPC and CoAP.
